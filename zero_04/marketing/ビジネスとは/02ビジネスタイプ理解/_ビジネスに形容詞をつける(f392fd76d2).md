## 外部環境に対する適応力（変化への強さ・広がり）
### 持続的なビジネス
- 事業
  - 特定の目的 = 必要。コアバリューの理解に必要。
  - バリューチェーン = 無理なく利益を出せる構造が必要。いわゆる「筋肉質な経営」と同義。
- 経営
  - バリューシステムの理解 = 必要。コアバリューを理解し、事業の本質を見極める必要がある。
  - バリューチェーンの調整 = コアバリューでない部分を調整することで、事業の持続性を高めることができる。
### トレンドに乗ったビジネス
- 事業
  - 特定の目的 = 不要。トレンドに乗ることが目的であるため、特定の目的は必要ない。
  - バリューチェーン = 不要。コピペでよい。
- 経営
  - バリューシステムの理解 = たぶん不要。
  - バリューチェーンの調整 = たぶん不要。
### スケーラブルなビジネス
- 事業
  - 特定の目的 = 必要。
  - バリューチェーン = コアバリューが属人的でないことが必要。「誰にでもできる」状況を作り出すことが必要。
- 経営
  - バリューシステムの理解 = たぶん不要。
  - バリューチェーンの調整 = 仕組みのコピペになるため、調整は難しく、持続可能性とトレードオフになりがち。（だんだん劣化する）
### 成熟したビジネス
- 事業
  - 特定の目的 = 不要
  - バリューチェーン = 調整済みで存在。固いビジネス。
- 経営
  - バリューシステムの理解 = 不要あるいは、暗黙知になっている。
  - バリューチェーンの調整 = 不要。業界全体が成熟しているため、調整の必要がない。ただし、技術革新や規制緩和などによる変化には注意が必要。
### 未熟なビジネス
- 事業
  - 特定の目的 = 必要
  - バリューチェーン = 荒削りでよく、最適化されてなくて構わないが、変化に強いことが重要。
- 経営
  - バリューシステムの理解 = 必須。既存の業界のニッチな部分や、技術革新・規制緩和などで空いた穴に参入するため、自身の立ち位置を正確に認知する必要がある。
  - バリューチェーンの調整 = 必要。業界全体が変化の途上にあるため、プレイヤー間の力関係や、価値を生み出すポイントが変わるため、調整が必要。
### 安定したビジネス
- 事業
  - 特定の目的 = 不要？
  - バリューチェーン = 「筋肉質な経営」とイコールではなく、複数の安定した収益基盤を持っていたり、外部変化への耐性などが必要。
- 経営
  - バリューシステムの理解 = 不要。安定した収益基盤のうえにあぐらをかくことができるから。
  - バリューチェーンの調整 = 安定した収益基盤の変化に応じた調整で十分。

### 利益率の高い「だけ」のビジネス
- 事業
  - 特定の目的 = 不要
  - バリューチェーン = 不要。どこかに無理をさせることで、利益だけを上げることができる。ただし、持続性とのトレードオフがある。
- 経営
  - バリューシステムの理解 = 必要。どこに無理をさせることができるか？を見極める必要がある。
  - バリューチェーンの調整 = 必要。無理をさせる部分を調整することで、利益率を上げることができる。
